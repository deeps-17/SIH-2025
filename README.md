# 🐄 Cattle Breed Recognition System

A comprehensive solution for cattle breed identification, developed as part of the **SIH2025** initiative. This repository combines a production-ready web application with a deep-dive research and evaluation suite.

---

## 📁 Repository Modules

### 🚀 [functional-app](./functional-app)
The core production module. It features a **FastAPI backend** and a **React frontend**, serving a MobileNetV2 model for real-time breed classification and characteristic analysis.

### 📊 [model-analysis](./model-analysis)
The research and development suite. This directory contains the **Jupyter notebooks**, data scarcity experiments, and model comparison results (CNN vs. Random Forest vs. Transfer Learning) that informed the final production model.
